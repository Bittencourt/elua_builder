verifyDate (date, dateFormat)
Finds out if a date is valid and returns day, month and year in separate elements on a table.

Parameters

  • date: the date that eyou want to validate
  • dateFormat: the format of the date you're giving on date

Usage

  • You will be using d for day, m for month and y for year
  • The date can be in any format you want as long as you put the format you're using in dateFormat:
    dd/mm/yyyy will work with 02/12/2000, but you'll need d/mm/yyyy to work with 2/12/2000.
  • You don't need to use any separators if u dont want too:
    ddmmyyyy will work with 02122000
  • Date month and year can be sent in any order as long as it's on the same order as the parameter:
    yyyy/mm/dd will work with 2000/12/02

  • Example: 
     dateTable = {}
     date = "02/12/2000"; dateFormat = "dd/mm/yyyy"
     flag, message, dateTable = verifyDate(date, dateFormat)
     print(flag, message)						--> true	Valid date
     print(dateTable.day, dateTable.month, dateTable.year)		--> 2	12	2000

Return values:

  1. false for invalid date and true for valid date
  2. a string with "valid date" or the reason why the date is invalid
  3. a table with the value of day in .day, month in .month and year in .year
